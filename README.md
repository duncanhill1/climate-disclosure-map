# Climate Disclosure Map

A mock web app that helps users explore climate disclosure requirements across the world through an interactive map.

## Concept
The homepage should feature a large world map. Countries or jurisdictions with climate disclosure requirements are highlighted in blue. On hover, users should see a popup with:
- jurisdiction name
- disclosure title
- short summary
- link to the official government source

## Special requirement
The app must support both:
- country-level disclosures
- subnational disclosures such as California

## Intended stack
- Next.js
- TypeScript
- Tailwind CSS
- react-simple-maps

## Initial scope
Phase 1 should include:
- homepage
- full-width interactive world map
- hover tooltip
- legend
- placeholder disclosure data
- support for country and state-level entries

## Notes
This repo is intended as a mockup first, but should be structured cleanly enough that it could be expanded later.
