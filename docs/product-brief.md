# Product Brief

## Working title
Climate Disclosure Map

## Purpose
Create an educational and visually strong web experience that helps users understand where climate disclosure regimes exist across the globe.

## Primary user experience
A user lands on the homepage and immediately sees a world map as the focal point.

Jurisdictions with climate disclosure requirements are highlighted in blue.

When the user hovers over one of these jurisdictions, a popup appears with:
- jurisdiction name
- disclosure name
- short explanation
- official government link

## Example
California should be supported as a subnational jurisdiction, not just countries.

## Phase 1 goals
- Build a polished front-end mockup
- Use placeholder data until the real list is provided
- Make the map the hero feature
- Ensure the structure can scale later

## UI guidance
- Minimal, modern, professional
- Clear spacing
- Strong visual hierarchy
- Blue highlight for disclosure jurisdictions
- Grey for areas with no disclosure data shown
- Clean tooltip or hover card

## Future-friendly design
The structure should allow:
- more jurisdictions later
- filtering later
- click-to-open side panel later
- article/detail pages later
